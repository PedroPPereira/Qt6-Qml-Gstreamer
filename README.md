# Qt6-Qml-Gstreamer

## Description
This is a simple QtQuick application to demostrate integrating Gstreamer into Qt6 qml UI.

## Build Info
- Qt 6.8.2 MSVC2022 64bit
- Gstreamer 1.22.12 (Complete installation of runtime and development package)
